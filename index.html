Quick Start

armazon.css is a lightweight and supersimple CSS3 Boilerplate built on LESS that will make a lot easier to start your Web application.
It provides a proper cross-browser consistency on all the common HTML elements display, overrides the default browsers ugly looking styling and with the adaptive grid system classes given, armazon.css is the perfect starting point tool to create and develop your responsive web app design.
In order to have armazon.css up and running on your site, just follow these simple steps:

  1. Include the armazon file onto your web document.

    • By using a copy of armazon.css, you can download the file from the armazon.css project main repository : armazon.min.css
    • Or using the hosted file using the CDN kindly provided by rawgit.com 
    <!-- Using a downloaded copy: -->
<link rel="stylesheet" type="text/css" href="path/to/your/armazon.min.css" />
<!-- Using CDN's hosted file: -->
<link rel="stylesheet" type="text/css" href="https://cdn.rawgit.com/alterebro/armazon.css/v0.1.0/styles/armazon.min.css" />

  2. Set the viewport meta tag inside the head of your document.

    <meta name="viewport" content="width=device-width, initial-scale=1" />

  3. That's all. Now, you can start your development, here's a simple template to get you started : 

    <!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
	<title>...</title>
	<meta name="viewport" content="width=device-width, initial-scale=1" />
	<link rel="stylesheet" type="text/css" href="https://cdn.rawgit.com/alterebro/armazon.css/v0.1.0/styles/armazon.min.css" />
	<style type="text/css">
	/* Start your CSS stylying here... */
		body {
			width: 90%;
			max-width: 980px;
			margin: 30px auto;
		}
	</style>
</head>
<body>
	<h1> Title of the Website </h1>
	<p> Text Paragraph ... <br /> Lorem ipsum dolor sit amet... </p>
</body>
</html>

The Grid

Adaptive Grid System

The grid works by simply creating a container with a .row class and putting within it columns with the desired width. The class names for the columns are intuitive and easy to remember, they are named:
.one-fourth, .one-sixth, .two-thirds ... 

<!-- Two columns, each one is half wide  -->
<div class="row">
	<div class="column-one-half">one-half</div>
	<div class="column-one-half">one-half</div>
</div>

Offsetting

Columns inside the row grid may have a displacement or offset that can be set by adding another class name following the form: shift-SIZE. For instance, some offsetting classnames would be:
.shift-one-sixth, .shift-one-fourth, .shift-one-third... 

<!-- A half-sized column with left displacement by 1/2 -->
<div class="row">
	<div class="column-one-half shift-one-half">by 1/2</div>
</div>
