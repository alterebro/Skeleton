armazon.css

Built on LESS, armazon.css is a simple and lightweight CSS3 Boilerplate based on normalize.css and Skeleton. It only styles the standard HTML tags and includes a responsive grid system semantically named using classes as : 'one third', 'one fourth', 'two thirds'...
